IDM 241: Microinteractions
Alpha assignment - Anna Leong

The trigger for this microinteraction would be when users tap (or click) the search icon.

After the trigger, the search icon bounces down and extends into a search bar. Additionally, text shows up to further indicate users to search.

The user will see all of the rules applied. The animation of the search icon expanding lets the user know that this is a search field/function.

This microinteraction does not loop. It only goes once each time the microinteraction is triggered.
