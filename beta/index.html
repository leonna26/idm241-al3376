The trigger for this microinteraction would be when the user hovers over the search icon button.

After the trigger, the search bar with a text input field opens horizontally with "Search" shown, allowing the user to type.

The user will visually see the rule applied.

This microinteraction does not loop, the mode is the "search" text with the text input field shown.
