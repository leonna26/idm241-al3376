Search

Triggers

The trigger for this microinteraction would be when the user clicks or taps the search icon.

Rules

After the trigger, the search icon bounces down, the search bar with a text input field opens horizontally with "Search" shown, allowing the user to type. If the user wants to cancel his search, he can click the search icon for the the bar to close and go back up to the original position.

Feeback

The user will visually see the rules applied and be able to interact by typing.

Loops and Modes

This microinteraction does not loop, the mode is the "search" text with the text input field shown.
