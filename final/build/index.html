Search

Triggers

The trigger for this microinteraction is when the user clicks or taps the search icon.

Rules

After the trigger, the search icon bounces down. The search bar with a text input field opens horizontally with "Search" shown when the user goes to type in the field through hovering.

Feeback

The user will visually see the rules applied and be able to interact by typing. Additionally, the user will see a visual cue of the autofocus blinking indicating that the user can type.

Loops and Modes

This microinteraction does not loop, the mode is the "search" text with the text input field shown, and the autofocus blinking.
