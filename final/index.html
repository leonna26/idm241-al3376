Source example

The trigger for this microinteraction would be when the user clicks or taps the search icon.

After the trigger, the search icon bounces down, the search bar with a text input field opens horizontally with "Search" shown, allowing the user to type. After the user enters something into the text field, the bar closes and goes back up.

The user will visually see the rules applied and be able to interact by typing.

This microinteraction does not loop, the mode is the "search" text with the text input field shown.
